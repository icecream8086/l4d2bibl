# 如何添加模板

## 文章类型

在文件开始时添加，主要帮助快速索引，tag可以是插件名等，category尚未细分，可能需要细分一下，但是插件类模板一定存在plugin

不添加此条目无法被索引

> 注:可能有bug，预览文章可能需要手动重启终端

文档类:

```yaml
---
category:
  - plugin
tag:
  - base
  - dev
archive: true

---
```

其他:

```yaml
---
date: 2022-01-12
category:
  - plugin
tag:
  - base
  - dev
---
```

## 填写模板

参考 `sourcemod插件模板` 编撰，有条件的话尽可能遵守`cc-by-sa` 协议注明参考条目来源

测试用例不是很重要，短时间可以不处理，格式无特殊要求，只是个填写建议

markdown表格推荐压榨ai解决，但是需要检查ai输入并小心求证
